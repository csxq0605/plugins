# outreach (Claude Code Plugin)

学术套磁全流程自动化 — 材料解析、教授调研、报告生成、邮件撰写。

## 安装

```bash
claude install-plugin github:csxq0605/plugins
```

## 使用

```bash
/outreach "这是我的CV"                    # 解析材料
/outreach "调研 MIT CS"                   # 调研教授
/outreach "生成报告 MIT CS"               # 生成 HTML 报告
/outreach "生成邮件 MIT CS"               # 生成套磁邮件
```
